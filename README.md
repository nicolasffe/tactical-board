# Tactical Board ⚽

Um quadro interativo para desenhar táticas e criar estratégias de jogo.

## Como instalar e executar

Abra o seu terminal e execute os seguintes comandos, um após o outro:

1. Clonar o projeto para o seu computador:
git clone https://github.com/nicolasffe/tactical-board.git

2. Entrar na pasta do projeto:
cd tactical-board

3. Instalar as dependências necessárias:
npm install

4. Iniciar a aplicação:
npm run dev

Pronto! Agora é só abrir o seu navegador e acessar http://localhost:3000
